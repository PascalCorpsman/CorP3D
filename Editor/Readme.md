# Controls

| command | description |
| --- | --- |
| Left Mouse + Mouse Move | Camera / selected object Rotation 
| Right Mouse + Mouse Move | Camera / selected object Translation
| b | create a small box at (0,0,0)
| B | create a big box at (0,0,0)
